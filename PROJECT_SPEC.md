# Project: Voyager Tarot Creations リニューアル計画

## 1. プロジェクト概要
**概要:**
「Voyager Tarot Creations」の現行サイト（外部サービス）が契約満了となるため、新規にリニューアル制作を行う。
**最重要方針:** 掲載する文章・コンテンツは、現行サイト（https://www.voyagertarotcreations.com/）の内容を**そのまま流用する**こと。
お客様の要望は「文言は今のままで良いが、デザインだけを明るく良くしてほしい」という点にあるため、勝手なリライトは行わず、デザインと構成の刷新に注力する。

**納期:** 2026年3月13日（絶対厳守 / サーバー契約切れのため）
**ターゲット:** ボイジャータロットを学びたい人、資格を取りたい人、悩みを持つ女性。

## 2. 技術スタック (Tech Stack)
* **Framework:** Astro (Latest)
* **UI Library:** React (タロット機能、動的パーツに使用)
* **Styling:** Tailwind CSS
* **CMS:** MicroCMS (お知らせ、ブログ記事管理)
* **Hosting:** Firebase Hosting or Cloudflare Pages
* **Animation:** Framer Motion (Reactコンポーネント用) or Standard CSS

## 3. デザインガイドライン & 雰囲気
**コンセプト:**
「組織・スクール」としての硬さを取り除き、主宰者（青木咲羅様）の人柄が伝わる**「温かみ」「癒やし」「明るさ」**のあるデザインにする。
タロットカードの「濃いブルー」を活かしつつ、全体が暗くならないよう配慮する。

**リファレンス（お手本・雰囲気の参考）:**
以下のサイトのデザインや構成を参考に、日本の占い・セラピー系サイトの雰囲気を掴むこと。
* **[Voyager Tarot Japan](https://voyagertarotjapan.com/)** (カード機能の参考含む)
* **[栄養コンシェルジュ協会](https://eiyo-c.com/)** (構成・見せ方の参考)
* **[一般財団法人 日本プロスピーカー協会](https://www.labo-c.com/)** (信頼感・認定講座の見せ方)
* **[占いアカデミー](https://www.uranai-academy.jp/)** (スクールとしての明るさ・ポップさ)
* **[セラピーサロンAYURA](https://aoki-sakura.com/)** (姉妹サイト・既存HP / 配色や雰囲気のベース)

**配色設計 (Color Palette) - Tailwind Config案:**
* **Base:** ホワイト (`#ffffff`), オフホワイト (`#fcfcfc` - 既存HPの背景色に近い色)
* **Text:** ダークグレー (`#333333`), 明朝体での見出し
* **Accent:**
    * **Sakura Pink:** 既存HPのリンク色やアクセントに使用されているピンク。
    * **Tarot Blue:** ボイジャータロットの裏面（濃い青）が映える色味。※全体を青くしすぎないよう注意。

**レイアウトの特徴:**
* **PC表示時:** 既存HP同様、**「右サイドバー（追従型）」**を設置すること。
    * サイドバー内容：プロフィール写真、各メニューへのリンクバナー、お問い合わせボタン。
* **SP表示時:** ハンバーガーメニューおよび、下部固定フッター（CTA）などを検討。

## 4. サイトマップ & ページ構成
全5〜6ページ構成。

1.  **TOPページ (`/`)**
    * ヒーローエリア（温かい写真＋キャッチコピー）
    * News（MicroCMSから取得・3件表示）
    * About（ボイジャータロットとは / Creationsの想い）
    * Course Guide（講座の簡易紹介）
    * **★Feature: Today's Voyager（タロット1枚引き機能への導線）**
2.  **プロフィール (`/profile`)**
    * 代表挨拶（青木咲羅）
    * 認定ティーチャー/コーチ/リーダー紹介一覧（※ここを充実させたい要望あり）
3.  **講座紹介 (`/course`)**
    * リーダー認定講座
    * コーチ認定講座
    * ティーチャー認定講座
    * 価格表
4.  **お知らせ一覧 (`/news`)**
    * MicroCMSと連携した記事一覧
5.  **お知らせ詳細 (`/news/[id]`)**
    * 記事詳細ページ
6.  **お問い合わせ (`/contact`)**
    * フォーム（Googleフォーム埋め込み or 静的フォームサービス）
    * 予約システム（freee予約）へのリンクボタン

## 5. 機能要件 (Functional Requirements)

### A. ボイジャータロット1枚引き機能 (`<TarotDraw />`)
**要件:**
* Reactコンポーネントとして実装。
* 参考：[Voyager Tarot Japan](https://voyagertarotjapan.com/) のカード引き機能。
* 「カードを引く」ボタン → シャッフルアニメーション → カードが1枚選ばれる → 表返る → メッセージが表示される。
* **演出:** リッチなアニメーション（Framer Motion推奨）。

### B. お知らせ機能 (MicroCMS)
**要件:**
* エンドポイント: `news`
* フィールド: タイトル, 本文(リッチエディタ), 日付, カテゴリ, アイキャッチ画像。

### C. 共通レイアウト (`Layout.astro`)
**要件:**
* ヘッダー：ロゴ、ナビゲーション。
* 右サイドバー：PC閲覧時に常時表示（スクロール追従）。SPでは非表示。
* フッター：コピーライト、SNSリンク、住所等。
